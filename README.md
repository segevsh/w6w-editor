# W6W Editor

A visual workflow editor for designing and managing workflows with drag-and-drop components.

## Overview

W6W Editor is an open-source React component library that provides a visual canvas for creating, editing, and managing workflows. Built with TypeScript and modern React patterns, it offers an intuitive interface for workflow design.

## Features

- **Visual Canvas**: Drag-and-drop interface for workflow design
- **Component Library**: Pre-built workflow components
- **TypeScript**: Full type safety and IDE support
- **Extensible**: Plugin architecture for custom components
- **Modular**: Monorepo structure with multiple workspaces

## Tech Stack

- React
- TypeScript
- Redux (state management)

## Project Structure

This is a monorepo containing multiple related workspaces for the editor and its ecosystem.
